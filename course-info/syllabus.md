# CSI403: Full Stack RAG with Local LLM - Syllabus

## Course Description

หลักสูตรเชิงปฏิบัติการที่จะพาคุณเรียนรู้การพัฒนาระบบ **Retrieval-Augmented Generation (RAG)** แบบ **Full Stack** โดยใช้ **Local LLM** ที่รันบนเครื่องของตัวเอง ไม่ต้องพึ่งพา API Token จากภายนอก

---

## Learning Objectives

เมื่อจบหลักสูตรนี้ นักศึกษาจะสามารถ:
1. เข้าใจหลักการทำงานของ RAG และ LLM
2. พัฒนาระบบ RAG แบบ Full Stack ได้
3. ใช้งาน Local LLM (Ollama) ได้
4. Deploy ระบบด้วย Docker Compose
5. สร้าง CI/CD Pipeline ได้
