# Screenshots

ใส่ screenshots ของงานที่ทำในโฟลเดอร์นี้

## Screenshots ที่ต้องมี (10 คะแนน)

1. **swagger-ui.png** - หน้า Swagger UI (http://localhost:9000/docs)

2. **test-results.png** - ผลการรัน pytest (optional แต่แนะนำ)

## วิธีเปิด Swagger UI

```bash
# รัน API server
python src/api.py

# เปิด browser ไปที่
http://localhost:9000/docs
```

## วิธีถ่าย screenshot

### Windows
- กด `Windows + Shift + S` เพื่อเปิด Snipping Tool

### Mac
- กด `Cmd + Shift + 4` เพื่อเลือกพื้นที่
