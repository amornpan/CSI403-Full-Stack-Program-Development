# Documents

โฟลเดอร์นี้ใช้สำหรับเก็บ documents ที่ต้องการ process

## Task 3: เพิ่ม Documents ใหม่ (20 คะแนน)

นักศึกษาต้องสร้างไฟล์ `.md` ใหม่ **3 ไฟล์** ในโฟลเดอร์นี้

### ข้อกำหนด

1. **จำนวนไฟล์**: 3 ไฟล์
2. **นามสกุล**: `.md` (Markdown)
3. **ความยาว**: แต่ละไฟล์ต้องมี **อย่างน้อย 200 คำ**
4. **หัวข้อ**: เกี่ยวกับ AI/ML/RAG

### ตัวอย่างหัวข้อที่แนะนำ

- Vector Databases และการใช้งาน
- การทำ Text Chunking สำหรับ RAG
- Embedding Models และการเปรียบเทียบ
- การ Fine-tune Language Models
- Prompt Engineering Techniques
- การใช้ LLM ในองค์กร

### รูปแบบการตั้งชื่อ

```
document1.md
document2.md
document3.md
```

หรือตั้งชื่อตามหัวข้อ เช่น:
```
vector_databases.md
text_chunking.md
embedding_models.md
```

### ตัวอย่างโครงสร้างเนื้อหา

```markdown
# หัวข้อ

## บทนำ
[อธิบายภาพรวม 50-100 คำ]

## รายละเอียด
[อธิบายเนื้อหาหลัก 100-150 คำ]

## สรุป
[สรุปประเด็นสำคัญ 50 คำ]
```
