## วิธีถ่าย screenshot

### Windows
- กด `Windows + Shift + S` เพื่อเปิด Snipping Tool
- หรือกด `Print Screen` แล้ว paste ใน Paint

### Mac
- กด `Cmd + Shift + 4` เพื่อเลือกพื้นที่
- หรือ `Cmd + Shift + 3` เพื่อถ่ายทั้งหน้าจอ

### Terminal Screenshot
- สามารถใช้ terminal screenshot tools หรือถ่ายจากหน้าจอโดยตรง
