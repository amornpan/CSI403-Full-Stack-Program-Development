# Screenshots

ใส่ screenshots ของงานที่ทำในโฟลเดอร์นี้

## ตัวอย่าง screenshots ที่ควรมี

1. `git-log.png` - แสดงผล `git log --oneline`
2. `test-results.png` - แสดงผลการรัน pytest

## วิธีถ่าย screenshot

### Windows
- กด `Windows + Shift + S` เพื่อเปิด Snipping Tool
- หรือกด `Print Screen` แล้ว paste ใน Paint

### Mac
- กด `Cmd + Shift + 4` เพื่อเลือกพื้นที่
- หรือ `Cmd + Shift + 3` เพื่อถ่ายทั้งหน้าจอ
